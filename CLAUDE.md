# בינה בלב — מערכת ניהול פנים-ארגונית

## תיאור הפרויקט

מערכת ניהול פנים-ארגונית לארגון "בינה בלב", המאחדת תחת קורת גג אחת את ניהול הגיוס, הפרויקטים, ההדרכות וחממת הסטארט-אפים.

## Stack טכנולוגי

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Backend / DB:** Supabase (PostgreSQL + Auth + Storage)

## שפה ועיצוב

- שפת ממשק ראשית: **עברית**
- כיוון טקסט: **RTL** (right-to-left)
- יש לוודא `dir="rtl"` על אלמנט ה-`<html>` ו-`lang="he"`

## מודולים עיקריים

1. **גיוס** — ניהול מועמדים, תהליכי ראיונות וקבלה
2. **פרויקטים** — מעקב אחר פרויקטים פעילים וצוותים
3. **הדרכות** — תכנון וניהול תכניות הדרכה
4. **חממת סטארט-אפים** — ניהול חברות ומיזמים בחממה

## כללי עבודה

- כל הטקסטים בממשק בעברית בלבד
- שמות משתנים וקוד — באנגלית
- קומפוננטות ב-`src/components/`, דפים ב-`src/app/`
- Supabase types מגונרטים ב-`src/types/supabase.ts`
